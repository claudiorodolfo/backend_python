Recapitulação - Instruções Sequenciais

Fundamentos da programação sequencial em Python.

  • input(): entrada de dados do usuário
  • print(): saída de informações
  • Variáveis: armazenam valores (int, float, str, bool)
  • Operações matemáticas: +, -, *, /, //, %, **
  • Conversões: int(), float(), str()
  • Formatação: f-strings, .format()

Recapitulação - Instruções Condicionais

Estruturas de decisão para controle de fluxo.

  • if, elif, else: tomada de decisões
  • Operadores lógicos: and, or, not
  • Operadores de comparação: ==, !=, <, >, <=, >=
  • Estruturas aninhadas: condições dentro de condições
  • Validação: verificar entrada antes de processar
  • Expressões booleanas retornam True ou False

Recapitulação - Instruções de Repetição

Laços para executar código múltiplas vezes.

  • for: iteração sobre sequências
  • while: repetição baseada em condição
  • range(): gera sequências numéricas
  • break: sai do laço imediatamente
  • continue: pula para próxima iteração
  • Laços aninhados: loops dentro de loops
Slide 66 de 90

Recapitulação - Acumuladores e Contadores

Padrões comuns em laços de repetição.

  • Acumuladores: somam valores (soma += valor)
  • Contadores: incrementam quantidades (cont += 1)
  • Inicialização: começar com valor neutro
  • Uso em cálculos de médias, totais, máximos, mínimos
  • Essenciais para processar coleções de dados
Slide 67 de 90

Recapitulação - Strings

Manipulação e processamento de texto em Python.

  • Operações: concatenação (+), repetição (*)
  • Formatação: f-strings (f"Olá {nome}"), .format()
  • Métodos: .upper(), .lower(), .strip(), .split(), .join()
  • Indexação: acessar caracteres por posição (texto[0])
  • Slicing: extrair substrings (texto[0:5])
  • Validação: .isalpha(), .isdigit(), .isalnum()
Slide 68 de 90

Recapitulação - Coleções: Listas

Estruturas ordenadas e mutáveis para múltiplos valores.

  • Criação: lista = [1, 2, 3] ou lista = []
  • Métodos: .append(), .insert(), .remove(), .pop()
  • Indexação: lista[0] (primeiro), lista[-1] (último)
  • Fatiamento: lista[1:4], lista[:3], lista[2:]
  • List comprehension: [x*2 for x in range(10)]
  • Iteração: for item in lista:

Recapitulação - Operações com Listas

Manipulação avançada de listas.

  • Listas aninhadas: matrizes e estruturas complexas
  • len(): tamanho da lista
  • in: verificar se item existe (x in lista)
  • Operadores: + (concatenação), * (repetição)
  • Métodos: .sort(), .reverse(), .count(), .index()
  • List comprehension com condições
Slide 70 de 90

Recapitulação - Coleções: Tuplas

Estruturas ordenadas e imutáveis.

  • Criação: tupla = (1, 2, 3) ou tupla = 1, 2, 3
  • Imutável: não pode modificar após criação
  • Diferença vs lista: mais rápida, menos memória
  • Uso: dados fixos, coordenadas, retorno múltiplo
  • Desempacotamento: x, y = tupla
  • Operações: indexação, fatiamento, iteração
Slide 71 de 90

Recapitulação - Coleções: Dicionários

Estruturas chave-valor para dados associativos.

  • Criação: dict = {"chave": "valor"} ou dict = {}
  • Acesso: dict["chave"] ou dict.get("chave")
  • Métodos: .keys(), .values(), .items()
  • Modificação: dict["nova"] = valor, .update()
  • Remoção: del dict["chave"], .pop()
  • Iteração: for chave, valor in dict.items()
Slide 72 de 90

Recapitulação - Dicionários Avançados

Uso prático de dicionários em Python.

  • Dicionários aninhados: estruturas complexas
  • .get(): retorna None se chave não existe
  • .setdefault(): define valor padrão se não existe
  • Dict comprehension: {k: v*2 for k, v in dict.items()}
  • Útil para: APIs JSON, configurações, dados estruturados
  • Fundamental para desenvolvimento backend

Recapitulação - Funções

Organização de código em blocos reutilizáveis.

  • Definição: def nome_funcao(parametros):
  • Parâmetros posicionais: ordem importa
  • Parâmetros nomeados: nome_funcao(param=valor)
  • Valores padrão: def funcao(param=10):
  • Return: retorna valor ou None
  • Escopo: variáveis locais vs globais

Recapitulação - Funções Avançadas

Recursos avançados de funções em Python.

  • Docstrings: documentação com """ texto """
  • Múltiplos retornos: return a, b (retorna tupla)
  • Parâmetros arbitrários: *args, **kwargs
  • Funções como objetos: podem ser passadas como argumentos
  • Lambda: funções anônimas simples
  • Reutilização: DRY (Don't Repeat Yourself)

Recapitulação - Tratamento de Exceções

Gerenciamento de erros e situações excepcionais.

  • try: bloco de código que pode gerar erro
  • except: captura e trata exceções
  • else: executado se não houve exceção
  • finally: sempre executado
  • Tipos: ValueError, TypeError, IndexError, KeyError
  • Tratamento específico: except ValueError:
Slide 76 de 90

Recapitulação - Exceções Avançadas

Uso avançado de tratamento de exceções.

  • Captura múltipla: except (ValueError, TypeError):
  • raise: lançar exceção manualmente
  • Exceções customizadas: class MinhaExcecao(Exception)
  • Mensagens: except ValueError as e: print(e)
  • Boas práticas: tratar exceções específicas
  • Validação: evitar erros antes que aconteçam
