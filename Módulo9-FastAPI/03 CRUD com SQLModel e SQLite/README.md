# 03 - CRUD com SQLModel e SQLite

## Objetivo

Persistir dados em banco SQLite e organizar a aplicação por domínio.

## Conteúdo

- Modelos com SQLModel.
- Sessão de banco e dependência de conexão.
- CRUD completo.
- Separação entre `models`, `schemas` e `routers`.

## Resultado esperado

API persistente com operações de criar, listar, editar e remover registros.
